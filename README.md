This project is an interactive monthly financial dashboard built in Power BI to monitor revenue, profit, cash flow, and budget vs actuals. 
It includes 6 KPIs (Total Revenue, Total COGS, Net Cash, Gross Margin %, Budget Variance %, EBITDA %) and 7 visuals like Revenue & Profit Trend, Net Cash Waterfall, Budget vs Actuals,
and Receivables/Payables Aging.

Features:

Dynamic slicers: Date, Region, Product/Service, Receivables/Payables Aging, Budget Variance %

Drill-down enabled for trends

Professional themed design with clear KPI cards
